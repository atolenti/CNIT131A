1 What is the difference between XSLT 1.0 and XSLT 2.0?
XSLT 1.0 XSLT 2.0
No schema awareness Has schema awareness
(allows checking for valid input, temporary trees, and output documents)
Limited to 5 data types Offers more flexibility in data types
Regular expression is not supported Regular expression is supported
Supports only one result tree output Supports more than one result tree output
Does not support user-defined functions Supports user-defined functions using xsl:function
Complex grouping Simple grouping using xsl:for-each-group
Other Enhancements:
Better error handling because there are more error code definitions.
2 What is the difference between XPath 1.0 and XPath 2.0?

XPath 1.0 XPath 2.0
Limited to four expression types Offers more flexibility by having more expression types
Data model is based on node-sets Data Model is based on sequence
Functions and operators are limited There are new functions and operators available

Examples of new operators:
a. arithmetic operators (+, -, *, div, mod, idiv)
b. Boolean (and, or)

Examples of new function:
a. general string handling (lower-case, upper-case)
b. Regular expression (matches, replaces)
